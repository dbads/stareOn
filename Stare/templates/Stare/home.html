Personal Usage


Stunning PIE Charts

Very attractive and easy to understand PIE Charts enables users to understand their own application usage patterns.

Set Constraints

We have a constraint feature, which kind of enables users to schedule autoclose for any application after a fixed time limit. Isn't cool??

Default Reminders

Recommend users what to do at anytime on the basis of their previous activity pattern. This way we can never miss an important task. whoaa!!


Parental Controls


Watch your Kids

Parents can watch the activities of their child and can accordingly take some actions to turn them in right direction.

Auto Close

Parents can set auto close constraints on their childs devices to ensure that their boys don't over do any useless thing on their System.

synchronise All Systems

You can synchronise all your Systems. You just need to allow all the other devices to follow your activity logs.


Future Aspects


Reducing Page Faults

we can very much predict what user is most likely to do at a given time, so we can prefetch the pages for corresponding application in memory then.

E-Commerce API

We will provide an API to E-Commerce companies, using this API these companies can access the execlusive user data which they can use to perform some kind of profiling to target the better product to more appropriate audience.

Easy Large Scale Surveys

STARE allows large scale surveys with less effort and more ease.
